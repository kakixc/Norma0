КРИТИЧЕСКОЕ ПРЕДУПРЕЖДЕНИЕ О РИСКАХ

Использование программы Norma0 (автоматический спам) прямо нарушает правила Telegram и других чат-платформ.

❌ ВЫСОКИЙ РИСК БАНА: Программа может привести к блокировке вашего аккаунта.

Автор проекта не несет никакой ответственности за последствия. Вы используете программу исключительно на свой страх и риск.

Терминология и Пояснения

Термин Пояснение
Норма (сообщений) Минимальное или требуемое количество сообщений, которое необходимо набрать в групповом чате (например, за день/неделю) для сохранения членства, статуса или участия в розыгрышах.
Юзербот (Userbot) Программа, работающая от имени личного аккаунта Telegram (а не через API-ключ обычного бота). Именно это позволяет Norma0 отправлять и удалять сообщения.
Спам-цикл Автоматическая последовательность действий бота: отправить сообщение (обычно 1), отправить следующее (0), затем удалить оба сообщения. Цикл повторяется по заданному таймеру.
API ID / API Hash Уникальные ключи, которые вы получаете с сайта Telegram (my.telegram.org) для авторизации вашего личного аккаунта в программе Norma0.

О проекте

Norma0 — это уникальный юзербот для Telegram, разработанный для пользователей, которым необходимо быстро и автоматически набрать «норму» в групповых чатах. Программа примечательна тем, что была полностью создана нейросетью.

Возможности и механизм

  • Скорость: Быстрая отправка сообщений для эффективного набора нормы.
  • Чистота в чате: Автоматическое удаление отправленных сообщений (1 и 0) сразу после публикации.
  • Простое управление: Активация и деактивация бота прямо из чата с помощью команд.
  • Локальная работа: Работает с ваших учетных данных, которые сохраняются локально.

Установка и запуск (Краткая инструкция)

Полную пошаговую инструкцию можно найти в файле README.md.

  1. Скачайте: Загрузите исполняемый файл (.exe или .py) со страницы Релизы (Releases).
  2. Получите API-данные: На my.telegram.org создайте и получите ваши API ID и API Hash.
  3. Первый запуск: Запустите программу и введите полученные данные, ваш номер телефона и код 2FA (если есть).
  4. Авторизация: Введите код входа, который придет вам в Telegram.
  5. Конфигурация сохранится в файле bot_config.json.

Команды управления в чате

Команда Действие
норма 0 ВКЛЮЧАЕТ фарм сообщений. Бот начинает спамить и удалять.
норма стоп ВЫКЛЮЧАЕТ бота. Фарм останавливается.
