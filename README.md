#  Norma0: Проект, Написанный Нейросетью

Проект **Norma0** — это программа, **полностью написанная нейросетью**, предназначенная для людей, которым сложно **набрать норму** (количество сообщений) в чатах.

---

##  Предназначение

* **Цель:** Помочь пользователям достичь требуемой нормы сообщений в чатах телеграмма.
* **Механизм:** Программа **спамит** (автоматически отправляет большое количество сообщений) и **удаляет**.

---

## ⚠Важное Предупреждение

**ВНИМАНИЕ:** Использование программы **Norma0** несет в себе ** риск блокировки (бана)** вашего аккаунта в чатах.

* **Причина:** Действия программы (спам, удаление сообщений) нарушают правила большинства чат-платформ.
* **Предостережение:** **Будьте осторожны**
